# 10-Real-World - 实战项目

> 将所学知识应用到实际项目中

---

## 🎯 项目列表

### 项目 1: 智能客服机器人

**功能**:
- 自动回答常见问题
- 转接人工客服
- 记录对话历史

**技术栈**:
- ChatModel
- Memory
- RAG

### 项目 2: 文档问答系统

**功能**:
- 上传 PDF/Word 文档
- 基于文档内容问答
- 引用来源

**技术栈**:
- Document Loaders
- Vector Store
- RAG

### 项目 3: 代码助手

**功能**:
- 代码解释
- Bug 修复建议
- 代码生成

**技术栈**:
- Code LLM
- Tools
- Agent

### 项目 4: 数据分析助手

**功能**:
- 数据查询
- 可视化生成
- 报告撰写

**技术栈**:
- Pandas Tool
- Code Interpreter
- Agent

---

## 📁 项目结构

每个项目包含：
```
project-name/
├── README.md          # 项目说明
├── requirements.txt   # 依赖
├── src/
│   ├── main.py       # 主程序
│   └── utils.py      # 工具函数
├── tests/            # 测试
└── examples/         # 使用示例
```
